# **DisTalk – MacOS App**

DisTalk ist eine native MacOS-App, die das Web-Interface von [DisTalk](https://distalk.eu) in eine echte App-Erfahrung verwandelt. Es nutzt volle MacOS-Integration.

---

## **Features**

* Optimiert für MacOS
* Taskleisten-Icon & Sicherheit
* Funktioniert garantiert
* Perfekt angepasst für den Desktop
